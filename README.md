Консольное приложение для заметок
Необходимо написать проект, содержащий функционал работы с заметками. Программа должна уметь создавать заметки, сохранять их, читать заметки списка, редактировать заметки, удалять заметки.

Условия задачи
Реализуйте консольное приложение для заметок с сохранением, чтением, добавлением, редактированием и удалением заметок. Заметка должна содержать идентификатор, заголовок, текст заметки и дату/время создания или последнюю измененную заметку. Сохранение заметок нужно делать в формате json или csv (разделение полей рекомендуется делать через точку с запятой). Реализация пользовательского интерфейса студент может сделать так, как ему удобнее, можно в виде параметров запуска программы (команда, данные), можно в виде запроса команды из консоли и последующего ввода данных, как-то еще, на усмотрение студента.

Решение проблемы
При запуске была написана программа, которая принимает аргументы для добавления, удаления, чтения и обновления данных в файле. Программа может принимать одну команду, так и несколько разных за один запуск. При сохранении или обновлении заметки ее дата добавляется/обновляется в формате: день/месяц/год-час:минуты. 

Для запуска программы: запустите main.py с необходимыми аргументами.
